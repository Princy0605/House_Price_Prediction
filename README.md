## Project Workflow

### 1. Data Understanding
- Loaded dataset and understood structure
- Identified numerical and categorical features

### 2. Data Preprocessing
- Handled missing (null) values using appropriate techniques
- Cleaned dataset for inconsistencies

### 3. Feature Engineering
- Encoded categorical variables (e.g., MSZoning, LotConfig) using encoding techniques
- Selected relevant features for prediction

### 4. Feature Scaling
- Applied feature scaling to normalize numerical values
- Improved model stability and performance

### 5. Model Building
- Implemented **Linear Regression model**
- Trained model on processed dataset

### 6. Model Evaluation
- Evaluated model performance using:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score

---

## Results
- The Linear Regression model was successfully trained on historical house data.
- After preprocessing and feature scaling, the model showed improved predictive performance.
- The system can reasonably estimate house prices based on input features.
